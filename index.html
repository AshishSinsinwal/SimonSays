Rules for the game

  • Press any key to start the game.
  • Watch the sequence of button flashes carefully.
  • Repeat the sequence by clicking the buttons in the same order.
  • The sequence gets longer with each level.
  • If you click the wrong button, the game ends.
  • Your score is the highest level you successfully complete.
  • Try to beat the highest score every time you play!
